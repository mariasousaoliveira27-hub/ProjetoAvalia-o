# ProjetoAvalia-o
https://www.figma.com/make/brHzhokokDdYUDwWI8QJzg/Art-Gallery-Page-Layout?t=2vLbSo9ijYKmbQYo-20&fullscreen=1
A Galeria de Arte Parisiense é uma aplicação web desenvolvida para exibição e comercialização de obras de arte. O sistema oferece uma interface elegante e intuitiva, permitindo que os usuários naveguem por diferentes categorias artísticas, visualizem informações detalhadas sobre as obras e consultem seus respectivos preços.
O projeto foi desenvolvido com foco na experiência do usuário, apresentando um design moderno inspirado em galerias de arte contemporâneas.
Objetivos do Projeto
Disponibilizar um catálogo digital de obras de arte.
Facilitar a navegação entre diferentes estilos artísticos.
Exibir informações relevantes sobre cada obra.
Proporcionar uma experiência visual agradável e responsiva.
Demonstrar conceitos de desenvolvimento front-end e design de interfaces.

Conclusão

A Galeria de Arte Parisiense é uma aplicação web voltada para a exposição digital de obras de arte, oferecendo uma experiência visual sofisticada e intuitiva. O projeto demonstra conhecimentos em desenvolvimento web, organização de interfaces e criação de sistemas responsivos, servindo como base para futuras expansões e integrações com funcionalidades de comércio eletrônico.
